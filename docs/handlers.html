(static, constant) exports.filterHandler

Filter the data according to the filter keyword. Filtering is not sensitive to case but do not use a normalizer function for accent or hyphens. Return unmodified data if keyword is empty. Has an exact search behavior for parts of the string being between double quotes. Builds the searchingKeywords array by spliting the string : 1) according to double quote to isolate fragment inside and outside of double quotes. 2) splitting the fragment outside of double quote according to spaces. Then make the fitlering research according to all elements of the searchingKeyword array.
